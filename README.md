# wyy-mobile
仿网易云音乐app

## 项目运行前，需在本地启一个服务

要启动的项目地址： https://github.com/Binaryify/NeteaseCloudMusicApi

```
git clone git@github.com:Binaryify/NeteaseCloudMusicApi.git // 下载

npm install // 安装

node app.js // 启动
```

## Project setup
```
git clone https://github.com/ddx2019/wyy-mobile.git
yarn install
```
